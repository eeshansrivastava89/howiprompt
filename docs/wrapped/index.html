Methodology

How It Works

1

Data Collection

Reads JSONL conversation logs from Claude Code (~/.claude/projects/) and parses the conversations.json export from Claude.ai.

2

Message Extraction

Filters to human-authored prompts only. Excludes system commands, tool outputs, and empty messages. Preserves timestamps for temporal analysis.

3

Metric Computation

Runs regex pattern matching across all prompts to count politeness markers, backtrack phrases, questions, and command patterns.

4

Persona Classification

Computes two composite scores (Engagement and Politeness) and maps them onto a 2×2 matrix to classify your prompting style.

Composite Scores:

Engagement = (Question + Backtrack) ÷ 2

Politeness = Index − (Command × 0.5)
